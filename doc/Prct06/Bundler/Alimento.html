    class Prct06::Bundler::Alimento

    Class that rapresents an Aliment which is described by a name, the grams of co2 emitted per kg of produced Aliment, the m^2 per year used in order to produce it, and then we have the energy informations which is described by the amount of carbs, proteins and fat contained in 100g of the Aliment.

    new(name, porcions=1, geiPerKg, terreno, carbs, proteins, lipidos) click to toggle source

    The initializer of the class has an attribute called porcions which is used to indicate if we want more than 100g of the Aliment. So, for example, if we wanted 200g of Chicken we would instantiate an object with porcions=2

    # File lib/prct06/bundler.rb, line 26
def initialize(name, porcions=1, geiPerKg, terreno, carbs, proteins, lipidos)
        @name = name
        @geiPerKg = porcions*geiPerKg
        @gei = (@geiPerKg / 1000) * porcions * 100
        @terreno = porcions*terreno
        @carbs = porcions*carbs
        @proteins = porcions*proteins
        @lipidos = porcions*lipidos
end

    get_energia() click to toggle source

    Method to get the total kcal of 'porcions'*100g of the Aliment- using the assumptions that: each gram of fat is equal to 9 kcal; each gram of protein is equal to 4 kcal; each gram of carb is equal to 4 kcal.

    # File lib/prct06/bundler.rb, line 68
def get_energia
        @lipidos * 9 + @proteins * 4 + @carbs * 4
end

    inidicator_energia() click to toggle source

    This method calculated the energy indicator, also a number which ranges from 0 to 3 it equals 1.0 if the kcal of the aliment is less than 670; it equals 2.0 if the kcal of the aliment is greater than 670 and less than or equal to 830; otherwise it will equals 3.0

    # File lib/prct06/bundler.rb, line 86
def inidicator_energia()
        e = get_energia()

        if e < 670
                return 1.0
        elsif e <= 830
                return 2.0
        else
                return 3.0
        end
end

    The nutricional indicator of an Aliment is described by a number from 0 to 3 which means that the food has a Low nutricional impact if this item is within 0 and 1; it has a Regular impact if it's within 1 and 2 or an High impact if it's withing 2 and 3.
